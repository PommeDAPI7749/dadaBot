# dadaBot

## Requirements

- Node ^18.X
- npm ^9.X

## Installation

#### Clone the repo
```sh
git clone https://github.com/PommeDAPI7749/dadaBot.git &&
cd dataBot
```

#### Create the .env file
```env
TOKEN = // Discord token
GUILD = 694968857684869280 // Dada's guild ID
CLIENT_ID = // Discord client ID
twitchAppId = // Twitch app ID
twitchSecret = // Twitch secret
```

#### Install dependencies
```sh
npm i
```

#### Launch app
```sh
node .
```
